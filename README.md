# 全附属神秘时代4整合包介绍

## 前言

*   **运行环境**：本整合包**仅支持 Java 8**（可使用最新版本）。使用过旧的 Java 8 版本可能导致问题。
*   **创作灵感**：整合包受到一位好友的启发——“聚合已有的神秘附属，制作一个包含失传或经修复模组的全附属包”。由衷感谢他提供了许多宝贵的失传模组及修复版本。
*   **核心特色**：
    *   **全面汉化**：汉化了目前已知 **99%** 的神秘时代附属内容，堪称最全的中文神秘附属整合包，助您跨越语言障碍。
    *   **模组集合**：创造性地收录了 **格雷科技6、暮色森林、以太2、魔法金属、缤纷纪元、血魔法、神秘时代4、魔戒** 等大型模组，弥补了神秘4附属模组流失的现状。
    *   **高度自定义**：您可以将其视为一个“附属模板”，自由选用其中的模组来构建属于自己的整合包。
    *   **便利功能**：内置整合包作者制作的**模组列表和要素列表**，方便查阅与故障排除。若觉得研究笔记过多，可通过放置相关文件实现**笔记自动解锁**。

## 已知问题

1.  **魔灵法柱**：使用异域要素强化神秘敌对生物时，若出现“瘟疫横行”前缀，会导致存档崩溃。
    *   **建议**：不要进行此操作。
    *   **修复**：用最近一次的存档备份覆盖。
2.  **巫媒仪式台**：Shift + 右键点击仪式台会直接导致游戏崩溃。
3.  **魔戒冲突**：魔戒的攻击间隔与神秘工匠的活化工作台使用间隔存在冲突。
    *   **解决方案**：在游戏配置 (`Config`) 中关闭魔戒自身的攻击间隔即可。
