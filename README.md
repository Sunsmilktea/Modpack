## 已知问题（使用时请注意）

1. **魔灵法柱 (Thaumic Infusion)**  
   使用「异域要素」强化神秘敌对生物时，有极小概率刷出前缀为“瘟疫横行”的实体，会直接导致存档崩溃。  
   → 建议不要用异域要素强化敌对生物  
   → 若已崩溃，请用备份中最近一次的存档覆盖当前存档即可恢复

2. **巫媒的仪式台 (Witchery Altar)**  
   Shift + 右键仪式台会直接崩溃  
   → 暂时无解，请不要 Shift 右键

3. **魔戒模组攻击间隔冲突**  
   魔戒自带的攻击冷却与神秘工匠 (Thaumcraft) 的活化工作台 (Arcane Workbench) 使用间隔冲突  
   → 解决方案：进入 `config/lotr/common.cfg`，将 `Enable Attack Cooldown` 改为 false 即可
