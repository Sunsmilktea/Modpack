# All the thaumcraft's mod 整合包介绍

## 前言

整合包只能使用 Java8 启动，可以是最新的 Java8。Java8 版本号太旧了可能会导致问题。

整合包受到一位好友启发而制作：“要不聚合已有的神秘附属来制作一个全附属包吧，包括那些失传的或者原先不能使用修复后可以使用的模组”。这位好友向我提供了很多失传的神秘时代 4 附属模组，并且也提供了一些附属模组的修复版本，在此由衷的感谢他。

这个整合包汉化了目前已知神秘附属的 99% 的内容，也可以说算是目前最全的神秘附属整合包，让你不再发愁看不懂英文怎么办。

你可以看作这是一个附属模板整合，创造性的收录了格雷科技 6，暮色森林，以太 2，魔法金属，缤纷纪元，血魔法，神秘时代 4，魔戒等，你完全可以使用包里的模组去组成你需要的整合包，补充了神秘 4 附属丢失的现状。觉得研究笔记太多，非常恐惧，也可以加相关文件下的笔记自动解锁。

作者启动完整包要 6 分钟，进入世界要 10 分钟。

整合包内置有整合包作者自己制作的模组列表和要素列表方便查阅和排除故障。

整合包二次转载，请注明原帖地址。

## 已知问题

- **魔灵法柱**：用异域要素强化神秘敌对生物出来“瘟疫横行”前缀会导致存档崩溃。建议不要使用。修复办法：覆盖最近一次的存档备份。
- **巫媒仪式台**：Shift 右键仪式台会直接崩溃。
- **魔戒与神秘工匠**：魔戒的攻击间隔与神秘工匠的活化工作台使用间隔存在冲突，可以通过在配置文件中关闭魔戒自身的攻击间隔来解决。
- **字体设置**：不能保存。主页面强制设置为 Unicode 字体，但重进游戏不会保存。
- **桶与特殊液体**：使用原版桶右键河水、污水、海水会造成崩溃。  
  - 能进存档：桶会消失。  
  - 不能进存档：覆盖最近一次的存档备份。
- **Thaumcraft Minus Thaumcraft**：后续版本失误加入了此模组，会在打开魔导手册时自动解锁所有研究。若有备份，下载 1.3 版本覆盖存档即可修复。
